## About
- This a show case of my skills with Vue 3. The project is a online dummy shop.

## Tech
- Vue 3, composition API, script setup
- Pinia
- Axios
- Naive-ui (component library)

- Rest API (dummy json)
- Login
- Basket
- Components
- Error handling
- Products: search, filter, sort
